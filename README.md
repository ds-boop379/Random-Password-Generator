# Random Password Generator

**Автор:** Максим Карнаух

Приложение с графическим интерфейсом для генерации случайных паролей с сохранением истории в JSON.

## Возможности
- Выбор длины пароля ползунком (от 4 до 64 символов)
- Включение/выключение цифр, букв, спецсимволов
- Кнопка генерации и поле вывода
- История сохраняется в `history.json` и отображается в таблице
- Валидация: проверка допустимой длины и выбора хотя бы одного набора символов
- Понятные сообщения об ошибках
